CORS Misconfiguration PoC

This demonstrates how a CORS misconfiguration on api.vimeo.com can lead to account takeover.

Step 1: Steal JWT from /upload/config

Step 2: Use stolen JWT to access /me endpoint

Step 3: Account Information Retrieved

How this attack works:

  1. The malicious page makes a request to api.vimeo.com/upload/config
  2. Due to CORS misconfiguration, the browser allows reading the response
  3. The JWT token is extracted from the response
  4. The stolen token is used to make authenticated requests to the /me endpoint
  5. Sensitive user information is exfiltrated

Vulnerable Request:

GET /upload/config? HTTP/1.1
Host: api.vimeo.com
Origin: https://attacker.com
Authorization: jwt [REDACTED]
